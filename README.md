## Photos & Favicon

- **Photos:** Drop your property photos into `src/assets/photos/`. The `Gallery` component will automatically import any images in that folder (jpg, png, webp, svg) and display them in filename-sorted order. Replace the sample SVGs in that folder with your real photos.
- **Favicon:** A simple favicon SVG (`public/favicon.svg`) with the stacked letters "RS" over "BC" has been added and is referenced in `index.html`.

After adding or replacing photos, run the dev server:

```bash
npm run dev
```

If you want a raster `favicon.ico` for older browsers, generate one from `public/favicon.svg` and place it in `public/`.
